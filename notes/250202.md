is $n$ prime?

- divide n by 2
  - if divisible $\rarr$ not prime
  - else $\rarr$ keep going
- divide n by 3
  - if divisible $\rarr$ not prime
  - else $\rarr$ keep going
...
- divide n by $\lfloor\sqrt{n}\rfloor$
  - if divisible $\rarr$ not prime
  - else $\rarr$ PRIME!

$\lfloor 23.235223\rfloor = 23$

$\lfloor -23.235223\rfloor = -24$
